# Contribuer a Klixa Companion

Merci de contribuer. Klixa Companion est publie sous licence Apache-2.0.

## Developpement

Prérequis : Node.js 22 ou une version LTS plus recente.

```bash
npm ci
npm test
npm start
```

Pour l'application Windows :

```bash
npm run desktop
npm run dist:win
```

Ouvrez une issue avant une modification importante. Une pull request doit etre
ciblee, documentee et accompagnee de tests lorsque le comportement change.

En soumettant une contribution, vous acceptez qu'elle soit distribuee sous la
licence Apache-2.0 du projet. Ne soumettez aucun secret, token, certificat,
contenu proprietaire ou code que vous n'avez pas le droit de redistribuer.
